Projeto que simula a Copa do Mundo.

> Escopo:
São 32 seleções;
Em cada partida, a pontuação funciona da seguinte forma:
* 3 pontos caso vença;
* 1 ponto em caso de empate;
* 0 ponto caso perca.

No final, a seleção com maior pontuação ganha a rodada.
